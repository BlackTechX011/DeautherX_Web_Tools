DeautherX web Tools

1.3 inch OLED Display 0.96 inch OLED Display

You can use DeautherX without display or button using Web Interface

Note
  • DeautherX web tools exclusively support I2C type displays and no other types.
  • It does not provide support for SPI displays or any other display types.
  • To flash other displays or utilize different board-compatible firmware, please use Arduino IDE.
How To Flash DeautherX Firmware to ESP8266-based Board
  • Select display type
  • Connect your ESP8266 board via USB
  • Click "Connect" and choose the serial port of your ESP
  • On Device Dashboard, click "Install"
How To Open Web-Based Serial Monitor
  • Select display type (It is not important to select the correct display type; it is just necessary for the program to run)
  • Connect your ESP8266 board via USB
  • Click "Connect" and choose the serial port of your ESP
  • On Device Dashboard, click "Logs and Console"
